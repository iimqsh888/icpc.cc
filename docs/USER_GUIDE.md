## üí± OTC Trading

### Create Buy Order

**Buy CPC from sellers**

**Steps:**
1. Visit OTC platform
2. Click "Create Order"
3. Select "Buy"
4. Enter:
   - Token amount (must be whole numbers)
   - Price per token (in BNB)
5. Pay: Total BNB + 0.001 BNB fee
6. Confirm transaction
7. Wait for sellers to fill

### Create Sell Order

**Sell CPC to buyers**

**Steps:**
1. Visit OTC platform
2. Click "Create Order"
3. Select "Sell"
4. Enter:
   - Token amount (must be whole numbers)
   - Price per token (in BNB)
5. Approve CPC tokens
6. Pay 0.001 BNB creation fee
7. Confirm transaction
8. Wait for buyers to fill

### Fill Orders

**Complete existing orders**

**For Buy Orders (you sell CPC):**
1. Browse buy orders
2. Click "Fill"
3. Approve CPC tokens
4. Confirm transaction
5. Receive BNB (minus 0.2% fee)

**For Sell Orders (you buy CPC):**
1. Browse sell orders
2. Click "Fill"
3. Pay BNB amount
4. Confirm transaction
5. Receive CPC tokens

### Cancel Orders

**Cancel your unfilled orders**

**Steps:**
1. Go to "My Orders"
2. Find order to cancel
3. Click "Cancel"
4. Confirm transaction
5. Receive refund

## üîí Security Tips

### Wallet Security

‚úÖ **DO:**
- Keep seed phrase offline
- Use hardware wallet for large amounts
- Verify contract addresses
- Check transaction details before confirming
- Use official links only

‚ùå **DON'T:**
- Share private keys/seed phrase
- Click suspicious links
- Approve unlimited token amounts
- Use public WiFi for transactions
- Store keys digitally

### Transaction Safety

1. **Always verify:**
   - Contract address
   - Transaction amount
   - Gas fees
   - Recipient address

2. **Start small:**
   - Test with small amounts first
   - Verify everything works
   - Then proceed with larger amounts

3. **Check BSCScan:**
   - Verify contract is verified
   - Check transaction history
   - Review contract code

## üêõ Troubleshooting

### Wallet Won't Connect

1. Refresh page
2. Unlock wallet
3. Check network (BSC Mainnet)
4. Try different browser
5. Clear cache

### Transaction Fails

1. Check BNB balance (for gas)
2. Check token balance
3. Increase gas limit
4. Check token approval
5. Wait and retry

### Can't Claim Airdrop

1. Verify BABT NFT ownership
2. Check if already claimed
3. Ensure on BSC Mainnet
4. Check contract balance

### Order Won't Fill

1. Check token balance
2. Approve tokens first
3. Verify order is active
4. Check order details
5. Ensure sufficient BNB
